1 평균

1.1 (1) 한국 사람의 체중은 평균 70kg, 표준편차 3인 정규분포를 따른다고 하자. 임의의 100명의 한국인의 데이터를 모았을 때, 이들의 체중에 대한 표본평균은 대충 얼마일까?

-> 한국인의 체중이 평균 70kg인 정규분포를 따른다면 100명의 표본을 모았을 때 표본평균도 대충 70kg 근처일 것이다.

1.2 (2) 한국 사람의 신장은 평균 165cm, 표준편차 10인 정규분포를 따른다고 하자. 임의의 100명의 한국인의 데이터를 모았을 때, 이들의 키에 대한 표본평균은 대충 얼마일까?

-> 위와 똑같이 한국인의 키가 평균 165cm인 정규분포를 따른다면 100명의 표본을 모았을 때 표본평균도 대충 165cm 근처일 것이라고 생각함.

2 회귀분석

2.1 (1) 선형회귀모형은 여러 개의 설명변수들의 선형결합을 이용해서 “무엇”을 추정하고자 하는 것일까?

-> 선형회귀모형은 여러 개의 설명변수들의 선형결합을 이용하여 종속변수의 연속적인 값(평균값)을 추정하고자 하는 것임.

2.2 (2) 로지스틱회귀모형은 여러 개의 설명변수들의 선형결합을 이용해서 “무엇”을 추정하고자 하는 것일까?

-> 로지스틱회귀모형은 여러 개의 설명변수들의 선형결합을 이용하여 특정 사건이 발생할 확률(이진 결과의 확률)을 추정하고자 하는 것임.

3 t검정

3.1 t검정(t-test)는 언제 쓰는가

-> 두 집단의 평균이 통계적으로 유의하게 다른지 확인하고자 할 때 사용

3.2 t검정을 위해 어떤 가정들이 필요한가? (명사형으로 나열. 길게 쓰지 말 것)

-> 정규성, 독립성, 등분산성

3.3 왜 t검정은 이름이 t일까?

-> 검정통계량이 t-분포(t-distribution)를 따르기 때문에 t검정이라고 부름.

3.4 t검정에서 왜 정규분포에 대한 가정이 필요할까?

-> 표본평균의 분포가 정규분포를 따른다고 가정해야 t-통계량이 t분포를 따른다는 이론이 성립하기 때문임.

3.5 일반적인 t검정과 paired t-test의 차이는?

-> 일반적인 t검정은 독립된 두 집단의 평균을 비교하고, paired t-test는 같은 대상에서 측정된 두 값의 차이의 평균을 비교함.

3.6 paired t-test의 예시를 3개 쓰시오.

다이어트 전 체중 vs 다이어트 성공 후 체중 , 약 복용 전의 혈압 vs 복용 후의 혈압 , 공부 전 시험 점수 vs 공부 후의 시험 점수

    3.8 t검정을 여러 번 수행했을 때의 문제점은?

    -> 여러 번 검정을 수행하면 1종 오류(Type I error)가 증가함.
    -> 실제로 차이가 없는데도 우연히 유의하다고 나올 확률이 커짐.

    3.9 위의 문제를 해결하기 위해 어떤 방법을 쓸 수 있을까?

    -> 다중비교 보정(multiple comparison correction)을 사용한다.
    -> Bonferroni correction이나 FDR(False Discovery Rate) 보정을 적용할 수 있다.
    -> 여러 변수를 동시에 고려하는 MANOVA 같은 다변량 분석을 사용할 수도 있다.

    3.10 위의 분석만 하면 틀렸다. 왜?

    -> 여러 변수에 대해 다중검정을 했지만 오류율 보정을 하지 않았음.

4 ANOVA

4.1 ANOVA는 무엇의 약자?

→ ANOVA는 Analysis of Variance(분산분석)의 약자.

4.2 ANOVA는 언제 쓰는가?

→ 세 개 이상 집단의 평균이 서로 차이가 있는지 검정할 때 사용.

4.3 ANOVA의 기본 가정 (명사형 나열)

→ 정규성, 독립성, 등분산성

4.4 t검정을 여러 번 해서 각 그룹을 비교할 수 있지 않을까? 왜 ANOVA를 쓸까?

→ 여러 집단을 t검정으로 반복 비교하면 1종 오류(Type I error)가 증가함.
→ ANOVA는 모든 집단을 한 번의 검정으로 비교하여 오류율을 통제할 수 있다는 장점이 있음.

4.5 왜 이름에 분산이 들어갈까?

→ ANOVA는 집단 간 차이를 분산의 형태로 분해하여 비교.
→ 전체 변동을 집단 간 분산과 집단 내 분산으로 나눔.
→ 이 두 분산을 비교하여 집단 평균의 차이가 유의한지 판단.

4.6 F검정이란 무엇인가?

→ F검정은 두 분산의 비율을 이용하여 집단 간 차이가 있는지 검정하는 방법.
→ ANOVA에서는 집단 간 분산을 집단 내 분산으로 나눈 F통계량을 사용.

4.7 왜 이름에 F가 들어갈까?

→ 검정통계량이 F분포(F-distribution)를 따르기 때문에 F검정이라고 함.

4.8 F검정에 대한 가정 (명사형 나열)

→ 정규성, 독립성, 등분산성

4.9 왜 ANOVA에서 F검정을 수행할까?

→ 집단 간 평균 차이는 집단 간 분산과 집단 내 분산의 비율로 판단할 수 있음.
→ 이 비율을 계산한 값이 F통계량.
→ F통계량이 충분히 크면 집단 평균이 서로 다르다고 판단.
