3 t검정

3.1 t검정(t-test)는 언제 쓰는가

-> 두 집단에 대해 관심있는 연속형 변수 값들의 관측값들의 평균이 통계적으로 유의하게 다른지 확인하고자 할 때 사용

3.2 t검정을 위해 어떤 가정들이 필요한가? (명사형으로 나열. 길게 쓰지 말 것)

-> 정규성, 독립성, 등분산성

  • 정규성: 잔차(residual)가 평균 0을 중심으로 한 정규분포를 따른다는 가정이다.
  • 독립성: 각 관측값(또는 잔차)이 서로 영향을 주지 않고 독립적으로 생성되었다는 가정이다.
  • 등분산성: 모든 수준의 독립변수에서 잔차의 분산이 일정하게 유지된다는 가정이다.

3.3 왜 t검정은 이름이 t일까?

-> 검정통계량이 t-분포(t-distribution)를 따르기 때문에 t검정이라고 부름.

-> 정규성을 가정하더라도 모집단 분산을 알 수 없으면 이를 표본으로 추정해야 하므로 추가적인 불확실성이 반영된 t분포를 사용하며, 따라서 z검정이 아닌 t검정을 적용함.

3.4 t검정에서 왜 정규분포에 대한 가정이 필요할까?

-> t검정은 표본평균을 기반으로 한 검정이므로, 표본평균의 분포가 정규성을 가져야 하고, 이를 위해 모집단(또는 오차)이 정규분포를 따른다는 가정이 필요함.

3.5 일반적인 t검정과 paired t-test의 차이는?

-> 일반적인 t검정은 독립된 두 집단의 평균을 비교하고, paired t-test는 같은 대상에서 측정된 두 값의 차이의 평균을 비교함.

-> paired t-test에서는 같은 대상에서 얻은 두 값의 차이를 계산한 뒤, 이 차이의 평균에 대해 다양한 가설을 설정할 수 있음.

  • 양측검정: 두 시점 간 변화가 있는지(차이가 있는지)를 확인할 때 사용함.

  • 단측검정: 변화의 방향(증가하거나 감소하는지)에 대한 사전 가설이 있을 때 사용함.

3.8 t검정을 여러 번 수행했을 때의 문제점은?

-> 여러 번 검정을 수행하면 1종 오류(Type I error)가 증가함.
-> 이러한 1종오류들이 누적되고 전체적으로 귀무가설이 참임에도 적어도 하나를 유의하다고 잘못 판단할 확률이 크게 증가하는 다중검정 문제가 발생할 수 있음.

3.9 위의 문제를 해결하기 위해 어떤 방법을 쓸 수 있을까?


-> Bonferroni correction이나 Benjamini-Hochberg 방법 등의 p값 보정을 적용을 함.

  • FWER (Family-Wise Error Rate): 여러 가설 검정에서 하나라도 잘못된 양성(제1종 오이 발생할 확률을 통제하는 방법, Bonferroni 보정이 대표적.

  • FDR (False Discovery Rate): 유의하다고 판단된 결과들 중 실제로는 잘못된 양성의 비율을 통제하는 방법, Benjamini-Hochberg 방법이 대표적.

3.10 위의 분석만 하면 틀렸다. 왜?

-> 여러 변수에 대해 다중검정을 했지만 오류율 보정을 하지 않았음.

-> 정규성, 등분산성, 독립성 가정이 충족되지 않으면 t검정의 p값과 결론의 신뢰성이 떨어지므로, 단순히 다중비교 보정 이전에 검정 자체의 타당성부터 확보해야 함.

  • 정규성: Shapiro–Wilk 검정이나 Kolmogorov–Smirnov 검정을 사용하거나, Q-Q plot을 통해 분포가 정규형태를 따르는지 확인.

  • 등분산성: Levene 검정이나 Bartlett 검정을 통해 집단 간 분산이 동일한지 평가.

  • 독립성: 연구 설계 단계에서 확보하는 것이 원칙이며, 시계열 데이터의 경우 Durbin–Watson 검정 등을 통해 자기상관 여부를 확인.

4 ANOVA

4.1 ANOVA는 무엇의 약자?

→ ANOVA는 Analysis of Variance(분산분석)의 약자.

4.2 ANOVA는 언제 쓰는가?

→ 세 개 이상 집단의 관심 대상이 되는 연속형 변수의 평균이 서로 차이가 있는지 검정할 때 사용.

4.3 ANOVA의 기본 가정 (명사형 나열)

→ 정규성, 독립성, 등분산성

4.4 t검정을 여러 번 해서 각 그룹을 비교할 수 있지 않을까? 왜 ANOVA를 쓸까?

→ 여러 집단을 t검정으로 반복 비교하면 1종 오류(Type I error)가 증가함.
→ ANOVA는 모든 집단을 한 번의 검정으로 비교하여 오류율을 통제할 수 있다는 장점이 있음.

4.5 왜 이름에 분산이 들어갈까?

→ ANOVA는 집단 간 차이를 분산의 형태로 분해하여 비교.
→ 전체 변동을 집단 간 분산과 집단 내 분산으로 나눔.
→ 이 두 분산을 비교하여 집단 평균의 차이가 유의한지 판단.

  • 전체 변동: 모든 점수들이 평균에서 얼마나 들쭉날쭉 퍼져 있는지를 말함.

  • 집단 간 변동: 각 집단의 평균끼리 서로 얼마나 떨어져 있는지를 본 것.

  • 집단 내 변동: 같은 집단 안에서 개별 값들이 얼마나 흩어져 있는지를 봄.

  • 만약 집단 간 차이가 집단 내 흔들림보다 훨씬 크면, 우연이 아니라 진짜 차이라고 볼 수 있음.

  • 반대로 집단 내 흔들림이 더 크면, 평균 차이는 그냥 우연처럼 보이는 것일 가능성이 큼.

4.6 F검정이란 무엇인가?

→ F검정은 두 분산의 비율을 이용하여 집단 간 차이가 있는지 검정하는 방법.
→ ANOVA에서는 집단 간 분산을 집단 내 분산으로 나눈 F통계량을 사용.

  • F값이 커진다는 것은 집단 간 평균 차이가 집단 내 변동에 비해 훨씬 크다는 의미.

  • 즉, 집단들 사이의 차이가 단순한 우연으로 보기 어렵다는 신호.

  • 따라서 평균이 서로 다를 가능성이 높다고 해석할 수 있음.

4.7 왜 이름에 F가 들어갈까?

→ 검정통계량이 F분포(F-distribution)를 따르기 때문에 F검정이라고 함.

4.8 F검정에 대한 가정 (명사형 나열)

→ 정규성, 독립성, 등분산성

4.9 왜 ANOVA에서 F검정을 수행할까?

→ 집단 간 평균 차이는 집단 간 분산과 집단 내 분산의 비율로 판단할 수 있음.
→ 이 비율을 계산한 값이 F통계량.
→ F통계량이 충분히 크면 집단 평균이 서로 다르다고 판단.

4.11 10)의 분석만 하면 틀린 이유

→ ANOVA는 어떤 집단들 사이에 차이가 있는지만 알려주고 어느 집단 간 차이인지 알려주지 않기 때문에 사후검정(post-hoc test)이 필요함.

  • ANOVA를 적용하기 전에는 각 집단의 정규성(Shapiro–Wilk 검정 또는 Q-Q plot), 등분산성(Levene 검정 또는 Bartlett 검정), 그리고 관측값의 독립성(연구 설계 또는 Durbin–Watson 검정)을 먼저 확인해야 함.

# 검사할 변수 목록
vars <- c("Sepal.Length", "Sepal.Width", "Petal.Length", "Petal.Width")

for (v in vars) {
  cat("\n====================================\n")
  cat("변수:", v, "\n")
  cat("====================================\n")
  
  # 1. 정규성 검정: 각 Species 그룹별 Shapiro-Wilk 검정
  cat("\n[1] 그룹별 정규성 검정 (Shapiro-Wilk)\n")
  split_data <- split(iris[[v]], iris$Species)
  shapiro_results <- lapply(split_data, shapiro.test)
  
  for (sp in names(shapiro_results)) {
    cat("\nSpecies:", sp, "\n")
    print(shapiro_results[[sp]])
  }
  
  # 2. 등분산성 검정: Bartlett 검정
  cat("\n[2] 등분산성 검정 (Bartlett test)\n")
  formula_txt <- as.formula(paste(v, "~ Species"))
  print(bartlett.test(formula_txt, data = iris))
  
  # 2-1. 등분산성 검정: Levene 검정
  cat("\n[3] 등분산성 검정 (Levene test)\n")
  print(leveneTest(formula_txt, data = iris))
  
  # 3. ANOVA 모델 적합
  fit <- aov(formula_txt, data = iris)
  
  # 4. Q-Q plot과 Residual plot
  par(mfrow = c(1, 2))
  qqnorm(residuals(fit), main = paste(v, "- Residual Q-Q Plot"))
  qqline(residuals(fit))
  
  plot(fitted(fit), residuals(fit),
       main = paste(v, "- Residuals vs Fitted"),
       xlab = "Fitted values", ylab = "Residuals")
  abline(h = 0, col = "red")
}

4.12 11) p-value 해석 기준

  • 정규성 (Shapiro-Wilk)
-    p \> 0.05 → 정규성 만족

-    p \< 0.05 → 정규성 위반
  • 등분산성 (Levene / Bartlett)
-    p \> 0.05 → 등분산성 만족

-    p \< 0.05 → 등분산성 위반

4.13 12) 변수별 분석 방법 선택

  • Sepal.Width
    → 정규성 O / 등분산성 O
    → ANOVA 적용

  • Sepal.Length
    → 정규성 O / 등분산성 X
    → Welch ANOVA 적용

  • Petal.Length
    → 정규성 O / 등분산성 X
    → Welch ANOVA 적용

  • Petal.Width
    → 정규성 X / 등분산성 X
    → Kruskal-Wallis 적용

4.14 13) 최종 요약

  • Sepal.Width → ANOVA

  • Sepal.Length → Welch ANOVA

  • Petal.Length → Welch ANOVA

  • Petal.Width → Kruskal-Wallis

4.15 14) Q-Q plot 해석

  • Q-Q plot은 정규분포 적합 여부를 시각적으로 확인하는 방법

  • 점들이 직선에 가까우면 정규성 만족으로 판단

  • 단, 주관적이므로 최종 판단은 Shapiro-Wilk 검정 사용

4.16 15) 그리고 우리가 ANOVA를 한다고 하면,

1) 정규성 검정 2) 등분산성 검정 3) ANOVA 검정
3개를 하는데, 이때 p값을 보정해야 할까요, 안 해야 할까요?

  • ANOVA는 한 번만 비교하는 거라서 점수(p값)를 따로 고칠 필요 없음

  • 정규성 검사, 등분산성 검사는 ANOVA를 해도 되는지 확인단계임.

4.19 Box plot 의미

  • 각 그룹의 중앙값(median) 비교 가능

  • 분산(퍼짐 정도) 확인 가능

  • 이상치(outlier) 확인 가능

4.20 ANOVA와 연결

  • 그룹 간 중앙값 차이가 보이면
    → ANOVA에서 유의할 가능성 높음

4.21 t-test와 연결

  • 두 그룹 간 박스 위치 차이가 크면
    → t-test에서 유의할 가능성 높음
